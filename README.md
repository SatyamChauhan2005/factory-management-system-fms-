# FMS-Tracker
🏭 Factory Management System (FMS) – Production Workflow Tracker

A web-based Production Tracking System designed to monitor and manage factory workflows in real-time.
This system digitizes manual processes and provides visibility into job progress, delays, and operator updates.

---

🚀 Overview

In many production environments, tracking workflow is manual, delayed, and lacks transparency.
This project solves that by providing a centralized system to track each stage of production.

---

🎯 Key Features

- 🔄 Step-wise Workflow Tracking
  Cutting → Bundling → Stitching → Kaaj → Thread Cut → Iron & Packaging

- 👨‍🏭 Operator Task Updates
  Workers can update task status directly from the web interface

- 📊 Live Dashboard
  Real-time insights into:
  
  - Pending Tasks
  - Work In Progress (WIP)
  - Delayed Jobs

- 🔍 Search & Filter
  Quickly find jobs using Job Card (JC) or article name

- ⏱ Time Tracking
  Tracks planned vs actual time and calculates delays/early completion

- 📁 Google Sheets Integration
  Acts as a backend database for storing and retrieving data

---

🛠 Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Google Apps Script
- Database: Google Sheets
- Deployment: Web App (Apps Script)

⚙️ How It Works

1. Admin enters production data into the system
2. Jobs are broken into step-wise tasks
3. Operators update task status via web interface
4. Data syncs with Google Sheets in real-time
5. Dashboard reflects live production status
